# 获取交互实例
## pickUser:
* 参数:

| 名称 | 类型 | 含义 |
| -- | -- | -- |
| uid | str | 用户唯一 id |

* 返回值:

| 名称 | 类型 | 含义 |
| -- | -- | -- |
| User | object | 用户交互性实例 |

返回值说明详见 [User]()
